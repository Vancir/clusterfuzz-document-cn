# Setting up fuzzing

以下页面将引导您设置模糊测试任务.

ClusterFuzz支持这两种类型的模糊测试: 覆盖率导向的模糊测试(使用[libFuzzer]以及[AFL])以及黑盒模糊测试. 查看[这个页面]以获悉这两种类型的比较.
